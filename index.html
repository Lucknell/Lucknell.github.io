    Time Attack

    Time Attack was a game that we developed for our Embedded Computing class.The game uses the Xilinx_XUPV5_LX110T Field Programmable Gate Array(FPGA) to take user input, display and run Time Attack. The game shows a hexadecimal number for the user to input using five buttons that have the values 1, 2, 4, 8, and 16. The user has to input the correct value before 15 seconds passes to finish the round. If the user succeeds they LCD of the FPGA will display great job, should the user not complete the task in the allotted time the game will interrupt the user and tell them that they have failed.
